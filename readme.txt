== Description ==

EN: Easy paging navigation for your blog using jQuery carousel.

RU: Удобная постраничная навигация с использованием JQuery carousel.

EXAMPLE: [wp-skins.info](http://wp-skins.info/ "wp-skins.info").


== Development Blog ==

[Truper WordPress Blog](http://wp-skins.info/ "Truper WordPress Blog")

== Installation ==

EN: Activate the plugin. Put in the right place the template function &lt;?php znavcarousel(); ?&gt; or &lt;?php znavcarousel(500); ?&gt;, where 500 - width in pixels.

RU: Активируйте плагин. Вставьте в нужном месте шаблона функцию &lt;?php znavcarousel(); ?&gt;
Если Вы хотите изменить ширину навигации, то используйте &lt;?php znavcarousel(500); ?&gt;, где 500 - ширина в пикселях.
